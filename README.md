# AssigmentTracker
This is a java application to help students track their homework and when it is due!

### Vision Product Statement 

For any student, who needs help managing and staying organized when working on assignments and tasks. Assignment Tracker is a productivity manager that allows easy management of tasks and assignments to help track progress. Unlike other apps which are targeted towards a wider audience this focuses on the needs of a student.


### Download Instruction


To download the application installer for your operating system:

1. Navigate to the Releases section located on the right side of the product page.

2. Under Releases, you will see three available download options:
   - macOS – AssignmentTracker-Installer.dmg
   - Windows – AssignmentTracker-Windows.msi
   - Linux – assignmenttracker_1.0.0_amd64.deb

3. Click the correct installer file for your operating system.
   
4. Once the download completes, locate the file in your Downloads folder.
   
5. For macOS (.dmg):
  - Double-click the .dmg file to open it
  - Drag the application icon into your Applications folder
  - Launch the app from the Applications folder or Launchpad

6.For Windows (.msi):
  - Double-click the .msi file to begin installation
  - Follow the on-screen setup instructions
  - After installation, find the app in your Start Menu or on your Desktop
    
7. For Linux (.deb):
  - Double-click the .deb file to open with your package installer (e.g., Ubuntu Software Center)
  - Click Install and enter your password if prompted
  - Launch the app from your applications menu
    
8. After installation, a double-clickable app icon will appear on your desktop (or in your applications list). Run it like any other application on your system.
